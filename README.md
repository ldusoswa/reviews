# Sim Racing Hardware Reviews

A collection of in-depth sim racing hardware reviews and scripts for video content creation.

## Overview

This repository contains review scripts and content for various sim racing hardware products including:

- **Pedals** - Brake, throttle, and clutch pedal reviews
- **Wheelbases** - Direct drive and belt-driven wheelbase reviews
- **Steering Wheels** - Wheel rim and button box reviews
- **Shifters & Handbrakes** - Sequential and H-pattern shifter reviews
- **Rigs** - Sim racing cockpit and rig reviews
- **Accessories** - Haptic feedback, dashboards, wind simulators, and more

## Structure

```
reviews/
├── Pedals/              # Pedal system reviews
├── Wheelbases/          # Wheelbase reviews
├── Steering Wheels/     # Wheel rim reviews
├── Shifters & Handbrakes/  # Shifter and handbrake reviews
├── rigs/                # Sim rig reviews
├── In Progress/         # Work in progress reviews
├── Blocked/             # On-hold reviews
└── [Individual files]   # Standalone product reviews
```

## Content Format

Each review file contains:
- Product overview and features
- Compatibility information
- Hardware specifications
- Software requirements
- Personal experience and testing
- Pricing and availability
- Purchase links (affiliate and direct)
- Final thoughts and recommendations

## Usage

These scripts are used for creating video content and written reviews for sim racing hardware. Content is organized by product category for easy reference and updates.
